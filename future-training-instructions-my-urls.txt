  Run Training in Colab

  1. Open https://colab.research.google.com
  2. File > Upload notebook → training/finetune.ipynb
  3. Runtime > Change runtime type → T4 GPU
  4. Run all cells — dataset loads automatically from HuggingFace

  No file uploads needed. The notebook will:
  - Load 89k samples directly from sachinbkale27/genetics-qa
  - Train for 1 epoch (~5,500 steps)
  - Save and download LoRA adapters

  Important: You should also rotate your HuggingFace token since it was shared in this conversation. Go to https://huggingface.co/settings/tokens and create a new one.
